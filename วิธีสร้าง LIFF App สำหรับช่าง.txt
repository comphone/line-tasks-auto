1. วิธีสร้าง LIFF App สำหรับช่าง และตั้งค่า Environment Variable
ขั้นตอนนี้สำคัญที่สุดเพื่อให้ฟีเจอร์ "แจ้งเตือนงานใกล้เคียง" ทำงานได้ คุณต้องทำเพียงครั้งเดียวครับ

ขั้นตอนที่ 1: เข้าสู่ LINE Developers Console
ไปที่เว็บไซต์ LINE Developers Console และล็อกอินเข้าสู่ระบบ

ขั้นตอนที่ 2: ไปที่ Channel ของคุณ
เลือก Provider ของคุณ

คลิกเข้าไปใน Channel ประเภท "LINE Login" ที่คุณใช้งานอยู่ (ไม่ใช่ Channel "Messaging API")

ขั้นตอนที่ 3: สร้าง LIFF App ใหม่
ในหน้า Channel ของคุณ ไปที่แท็บ "LIFF"

กดปุ่ม "Add" เพื่อสร้าง LIFF App ใหม่

กรอกข้อมูลดังนี้:

LIFF app name: ตั้งชื่อที่เข้าใจง่าย เช่น Technician Location Update

Size: เลือกเป็น Full

Endpoint URL: https://<ชื่อแอปของคุณ>.onrender.com/liff/technician/update_location

สำคัญ: ให้เปลี่ยน <ชื่อแอปของคุณ> เป็นชื่อแอปพลิเคชันของคุณบน Render.com เช่น https://line-tasks-auto.onrender.com/liff/technician/update_location

Scopes: ติ๊กเลือก profile และ openid

Scan QR: เลือกเป็น On

กดปุ่ม "Add" เพื่อบันทึก

ขั้นตอนที่ 4: คัดลอก LIFF ID
หลังจากสร้างเสร็จ ระบบจะแสดงหน้าข้อมูล LIFF App ใหม่

ให้คุณคัดลอก LIFF ID เก็บไว้ (ตัวอย่าง: 2007690244-AbCdEfGh)

ขั้นตอนที่ 5: ตั้งค่า Environment Variable บน Render.com
ล็อกอินเข้าสู่ Render.com

ไปที่ Dashboard และเลือก Service ของคุณ

ไปที่เมนู "Environment"

ในส่วน "Environment Variables" กดปุ่ม "Add Environment Variable"

ตั้งค่าดังนี้:

Key: LIFF_ID_TECHNICIAN_LOCATION

Value: วาง LIFF ID ที่คุณคัดลอกมาจากขั้นตอนที่ 4

กด "Save Changes"

Render จะทำการ Deploy แอปของคุณใหม่โดยอัตโนมัติ รอจนกว่าจะขึ้นสถานะ "Live"

ขั้นตอนที่ 6: นำ LIFF URL ไปให้ช่างใช้งาน
กลับไปที่หน้า LINE Developers Console

ในหน้า LIFF App ของช่างที่คุณเพิ่งสร้าง จะมี LIFF URL อยู่ (ตัวอย่าง: https://liff.line.me/2007690244-AbCdEfGh)

ส่งลิงก์นี้ให้ช่างของคุณ เพื่อใช้ในการอัปเดตตำแหน่ง

2. วิธีใช้งานฟีเจอร์การแจ้งเตือน
2.1 การแจ้งเตือนทั่วไป (แจ้งเตือนนัดหมาย, ติดตามลูกค้า)
การทำงานส่วนนี้จะเกิดขึ้นโดยอัตโนมัติตามเวลาที่คุณตั้งไว้ในหน้า "ตั้งค่าระบบ"

แจ้งเตือนนัดหมาย: ระบบจะส่งสรุปงานของ "วันนี้" ไปยังกลุ่ม LINE Admin/ช่าง ตามเวลาที่ตั้งไว้ในช่อง "แจ้งเตือนนัดหมาย (โมงไทย)"

ติดตามลูกค้า: ระบบจะส่งข้อความ Flex Message ไปยัง LINE ส่วนตัวของลูกค้าเพื่อสอบถามความพึงพอใจ หลังจากปิดงานไปแล้ว 1 วัน ตามเวลาที่ตั้งไว้ในช่อง "ติดตามลูกค้า (โมงไทย)"

เงื่อนไข: ลูกค้าจะต้องเคยสแกน "QR Code สำหรับลูกค้า" เพื่อเชื่อมต่อ LINE Account กับระบบก่อน

2.2 🔔 วิธีใช้งานฟีเจอร์ใหม่: "แจ้งเตือนงานใกล้เคียง"
ฟีเจอร์นี้จะทำงานได้ ต้องตั้งค่าทั้งฝั่ง Admin และให้ช่างอัปเดตตำแหน่งครับ

ขั้นตอนฝั่ง Admin (ตั้งค่าในเว็บ):

ไปที่หน้า "ตั้งค่าระบบ"

ใน Card "การตั้งค่าแจ้งเตือนบนมือถือ (LINE LIFF Popup)":

ติ๊ก "เปิดใช้งาน: แจ้งเตือนงานใกล้เคียง (สำหรับช่าง)"

กำหนด "รัศมีแจ้งเตือนงานใกล้เคียง (กม.)" เช่น 5.0

ใน Card "ข้อมูลร้านค้าและช่าง" > "จัดการรายชื่อช่าง":

กด "แก้ไข" ที่รายชื่อช่างที่ต้องการให้รับการแจ้งเตือน

สำคัญมาก: ตรวจสอบให้แน่ใจว่าได้กรอก "LINE User ID" ของช่างคนนั้นๆ ไว้อย่างถูกต้อง

กดบันทึก

ขั้นตอนฝั่งช่าง (อัปเดตตำแหน่ง):

ช่างเปิด LIFF URL ที่ได้รับมาจากคุณ (จากข้อ 1.6) ในแอป LINE

จะพบหน้าเว็บง่ายๆ ที่มีปุ่ม "อัปเดตตำแหน่งของฉัน"

เมื่อกดปุ่ม LINE จะขออนุญาตเข้าถึงตำแหน่งปัจจุบัน ให้ช่างกด "อนุญาต"

เมื่อสำเร็จ หน้าจอจะแสดงข้อความ "อัปเดตตำแหน่งเรียบร้อย!" และปิดตัวเองอัตโนมัติ

ช่างควรอัปเดตตำแหน่งทุกครั้งที่เริ่มงานในแต่ละวัน หรือเมื่อย้ายไปทำงานในโซนใหม่ เพื่อให้ข้อมูลเป็นปัจจุบันที่สุด

การทำงานอัตโนมัติ:

เมื่อมี "งานใหม่" ถูกสร้างขึ้นในระบบ และตำแหน่งของงานนั้นอยู่ในรัศมีที่กำหนดไว้ใกล้กับตำแหน่งล่าสุดของช่างคนใดคนหนึ่ง ระบบจะส่งการแจ้งเตือนส่วนตัวไปให้ช่างคนนั้นทันที

3. วิธีใช้งานฟีเจอร์ใหม่อื่นๆ
3.1 🗑️ การลบงานหลายรายการพร้อมกัน (Batch Deletion)
ไปที่หน้า "แดชบอร์ด" (หน้ารายการงาน)

คุณจะเห็น ช่องสี่เหลี่ยม (Checkbox) เพิ่มขึ้นมาหน้าแต่ละรายการงาน

ติ๊กเลือกงานที่ต้องการลบทีละรายการ หรือกด Checkbox ที่หัวตารางเพื่อ "เลือกทั้งหมด" ในหน้านั้น

เมื่อคุณเลือกอย่างน้อย 1 รายการ จะมี แถบสีดำ ปรากฏขึ้นที่ด้านล่างของจอ พร้อมบอกจำนวนที่เลือก

กดปุ่มสีแดง "ลบรายการที่เลือก" บนแถบสีดำ

ระบบจะถามเพื่อยืนยันอีกครั้ง กด "OK" เพื่อลบงานที่เลือกทั้งหมดอย่างถาวร

3.2 엑셀 การนำเข้า/ส่งออกข้อมูลอุปกรณ์ (Import/Export Equipment)
ไปที่หน้า "ตั้งค่าระบบ"

เลื่อนลงมาที่ Card ใหม่ชื่อ "จัดการข้อมูลอุปกรณ์ (Equipment Data)"

วิธีส่งออก (Export):

กดปุ่ม "ส่งออกเป็น Excel"

ระบบจะดาวน์โหลดไฟล์ equipment_catalog.xlsx ซึ่งเป็นข้อมูลอุปกรณ์ทั้งหมดที่คุณมีในระบบ ณ ปัจจุบัน

แนะนำ: ให้ใช้ไฟล์นี้เป็นต้นแบบในการแก้ไขข้อมูลก่อนนำเข้าใหม่

วิธีนำเข้า (Import):

เตรียมไฟล์ Excel ของคุณ โดยต้องมีคอลัมน์ชื่อ item_name, unit, price ให้ครบถ้วน

กดปุ่ม "Choose File" หรือ "Browse..." แล้วเลือกไฟล์ Excel ที่เตรียมไว้

กดปุ่ม "นำเข้า"

ข้อควรระวัง: การนำเข้าจะ ลบข้อมูลอุปกรณ์เก่าทั้งหมด และแทนที่ด้วยข้อมูลใหม่จากไฟล์ที่คุณอัปโหลดทันที
